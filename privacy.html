Política de Privacidad

Última actualización: 21 de marzo de 2026

GallOli ("la aplicación") es desarrollada y operada por Ivan Quiñonez. Esta política describe cómo se recopila, usa y protege la información cuando usas GallOli.

1. Información que recopilamos

GallOli recopila únicamente la información que tú ingresas directamente:

  • Nombres y datos de contacto de clientes (nombre, teléfono, dirección)
  • Ubicación geográfica de clientes (coordenadas GPS ingresadas manualmente o seleccionadas en mapa)
  • Registros de ventas, pedidos y pagos
  • Gastos operativos y datos contables
  • Registros de merma y precios diarios
  • Credenciales de Telegram (bot token y chat ID) para el sistema de backup automático

2. Cómo usamos la información

Toda la información recopilada se usa exclusivamente para:

  • Gestionar clientes, ventas y pedidos dentro de la aplicación
  • Generar reportes contables y estadísticas de negocio
  • Realizar backups automáticos a través de Telegram
  • Sincronizar datos entre dispositivos del mismo usuario

3. Almacenamiento de datos

Los datos se almacenan localmente en tu dispositivo usando IndexedDB. Opcionalmente, si configuras el backup a Telegram, los datos se envían cifrados a tu propio bot de Telegram. No almacenamos ningún dato en servidores propios de GallOli.

4. Compartir información con terceros

No vendemos, alquilamos ni compartimos tu información personal con terceros. Los únicos servicios externos utilizados son:

  • Cloudflare Pages: Hospedaje de la aplicación web
  • Telegram API: Solo si el usuario configura el backup automático
  • OpenStreetMap / Leaflet: Para visualización de mapas (sin recopilación de datos personales)

5. Permisos de la aplicación

  • Ubicación: Opcional, para asignar coordenadas a clientes en el mapa
  • Internet: Para sincronización, backup y carga de mapas
  • Almacenamiento: Para guardar datos localmente y exportar backups
  • Notificaciones: Para alertas de pedidos y recordatorios de backup

6. Seguridad

Las credenciales de Telegram se almacenan localmente en tu dispositivo. Recomendamos no compartir tu dispositivo con personas no autorizadas para proteger la confidencialidad de los datos de tu negocio.

7. Menores de edad

GallOli está diseñada para uso empresarial por adultos. No recopilamos intencionalmente información de menores de 13 años.

8. Cambios a esta política

Podemos actualizar esta política ocasionalmente. Te notificaremos sobre cambios significativos a través de la aplicación. El uso continuado de GallOli después de los cambios constituye aceptación de la nueva política.
